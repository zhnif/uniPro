## ğŸ› ï¸ Teknologi (Tech Stack)

Project ini dibangun dengan **Native Web Technologies** untuk performa maksimal dan kemudahan "Plug & Play" tanpa *dependencies* berat.

*   **HTML5**: Struktur semantik dan aksesibel.
*   **CSS3**:
    *   Variables (`:root`) untuk manajemen warna.
    *   Flexbox & Grid untuk layout responsif.
    *   Keyframe Animations untuk efek *pulse* dan *typing*.
*   **JavaScript (Vanilla ES6+)**:
    *   `IntersectionObserver` untuk animasi scroll performa tinggi.
    *   DOM Manipulation untuk kalkulator logika.
    *   Event Listeners untuk interaksi navbar dan slider.
*   **Icons**: [Feather Icons](https://feathericons.com/) (CDN).
*   **Fonts**: 'Poppins' (Google Fonts).
